# ChessVerse sound assets

Add production audio files here with these exact names:

- `chess_move.mp3` — real wooden chess move sound
- `chess_capture.mp3` — stronger wooden capture sound
- `chess_check.mp3` — short tense check alert
- `chess_checkmate_dark.mp3` — dramatic dark checkmate sound

The app sound service ignores missing files so preview builds remain stable until final audio files are added.
