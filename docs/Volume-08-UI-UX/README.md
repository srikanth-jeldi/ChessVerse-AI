## Sprint 1 UI Direction

The game screen should feel like a premium chess arena rather than a basic board demo.

Current implementation direction:

- Dark cinematic shell with brass, jade, ivory and walnut accents.
- Responsive layout for phones, tablets, desktop and web.
- Custom chess coin widgets instead of fragile font-dependent chess symbols.
- Board theme selector with Walnut, Jade and Classic options.
- Animated square selection, coin movement replacement and move history updates.
- AI control panel with coach toggle, strength slider, hint and analysis actions.
- Board coordinates, match clocks and compact status pills for a serious gameplay feel.

Quality bar:

- Every control must remain usable on mobile.
- Board and pieces must keep stable proportions across screen sizes.
- No feature text should cover the board or chess coins.
- Colors must support clear piece contrast in all board themes.
- Future piece assets can replace the widget-based coins without changing game state logic.
